## ğŸ”¥ Firebase Setup

### 1ï¸âƒ£ Create Firebase Project

1. Go to [Firebase Console](https://console.firebase.google.com/)

### 2ï¸âƒ£ Enable Authentication

1. In Firebase Console â†’ **Authentication** â†’ **Get Started**
2. Click **Email/Password** â†’ Enable â†’ **Save**
3. Add your admin user:
   - Go to **Users** tab â†’ **Add user**
   - Enter your email and password

### 3ï¸âƒ£ Set up Firestore Database

1. In Firebase Console â†’ **Firestore Database** â†’ **Create database**
2. Start in **Production mode**
3. Choose a location closest to your users
4. Add security rules:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /posts/{postId} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
```

### 4ï¸âƒ£ Enable Firebase Storage

1. In Firebase Console â†’ **Storage** â†’ **Get Started**
2. Start in **Production mode**
3. Add security rules:

```javascript
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /projects/{fileName} {
      allow read: if true;
      allow write: if request.auth != null && request.resource.size < 5 * 1024 * 1024;
    }
    match /blog/{fileName} {
      allow read: if true;
      allow write: if request.auth != null && request.resource.size < 5 * 1024 * 1024;
    }
  }
}
```

### 5ï¸âƒ£ Get Firebase Config

1. Go to **Project Settings** (âš™ï¸ icon)
2. Scroll down to **Your apps** â†’ **Web app** (</> icon)

```js
  match /databases/{database}/documents {
    match /posts/{postId} {
      allow read: if resource.data.published == true;
      allow read, write: if request.auth != null; // owner-only in UI
    }
  }
}
```

Recommended Storage security rules:

```js
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /projects/{fileName} {
      allow read: if true;
      allow write: if request.auth != null && request.resource.size < 5 * 1024 * 1024;
    }
    match /blog/{fileName} {
      allow read: if true;
      allow write: if request.auth != null && request.resource.size < 5 * 1024 * 1024;
    }
  }
}
```

With these rules, public visitors can read only published posts. Authenticated you (admin) can create, update, delete posts and see drafts.
